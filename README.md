# Sistema de Gestión Humana Rikimaka

## 📌 Descripción del proyecto
Este proyecto consiste en el desarrollo de una aplicación web para la gestión del talento humano de la organización Rikimaka. El sistema permitirá a los colaboradores registrar sus actividades diarias y solicitar vacaciones, mientras el departamento de Recursos Humanos podrá supervisar solicitudes y analizar métricas laborales para mejorar la planificación del personal.

## Alcance del Avance I
En el primer avance se realizó el diseño de la interfaz mediante wireframes de alta fidelidad, definiendo la estructura visual y la navegación del sistema.

## Alcance del Avance II
En este segundo avance se implementa la maquetación funcional del sistema utilizando HTML y CSS, tomando como base los wireframes desarrollados previamente.

Se desarrollaron las pantallas principales del sistema, con navegación entre ellas y una interfaz visual coherente con el diseño propuesto.

## Alcance del Avance III
Implementación JavaScript

## Funcionalidades implementadas en JavaScript
- Login simulado con validación de correo y contraseña
- Validación de formularios
- Registro dinámico de actividades
- Eliminación de actividades
- Envío de solicitudes de vacaciones
- Aprobación y rechazo de solicitudes

## Cómo ejecutar el proyecto
1. Descargar o clonar el repositorio.
2. Ubicar todos los archivos del proyecto en la misma carpeta o respetar las rutas configuradas.
3. Abrir el archivo `index.html` en el navegador.
4. Navegar entre las pantallas del sistema mediante los botones y enlaces disponibles.

## Estructura de archivos JavaScript
- `app.js`: contiene la lógica del frontend, validaciones, eventos y manipulación del DOM.
